# 🛡️ SCADA Guardian

Application desktop intelligente pour la **supervision, audit et sécurisation des systèmes SCADA industriels**.

Développée avec :
- ⚡ Vue.js (Frontend)
- 🖥️ Electron (Desktop App)
- 🚀 Node.js + Express (Backend)
- 🗄️ SQLite (Database)

---

# 📸 Aperçu

- Dashboard SCADA en temps réel
- Monitoring des données (température, pression, vitesse)
- Historique des données
- Backend intégré dans Electron
